Завдання на лабораторну роботу №1.

1. Налаштувати робоче оточення (ValentinaStudio, PostgreSQL)
2. Здійснити опис бізнес процесу для реалізації протягом вивчення дисципліни відповідно до отриманої теми (.txt file)
3. Відповідно до опису бізнес процесу створити ER діаграму структури бази даних (.pdf файл).
4. Створити відповідну базу даних та декілька користувачів з різними правами доступу до неї. (.sql файл)
5. Створити таблиці бази даних на локальному комп'ютері. (.sql файл)
6. Заповнити таблиці даними (mockaroo.com, .sql файл).
7. Створити репозиторій github під дисципліну та викласти результати виконання лабораторної роботи у папці lab1.
8. Заповнити таблицю Google docs з посиланнями на репозиторії.
