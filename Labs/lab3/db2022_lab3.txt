Завдання на лабораторну роботу №3.

1. Створити REST сервіс для роботи з базою даних
(виконання CRUD операцій), 
розробленої у попередніх лабораторних роботах.
2. Використати технології (СУБД: PostgreSQL, 
мова програмування на вибір: Java | Python | C#).
3. Для виконання CRUD операцій використовувати SQL 
(забороняється використовувати технологію ORM та подібні).
4. Надати інструкцію з використання сервісу в окремому файлі.
5. Викласти результати виконання лабораторної роботи (файли з кодом та документацію)
   у репозиторій GitHub у папку lab3.
